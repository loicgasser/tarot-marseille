# 🔮 Tarot de Marseille — Tirage des Arcanes Majeurs

Une application web de tirage de tarot de Marseille en français, avec les 22 Arcanes Majeurs et un tirage en croix de 4 cartes.

**[→ Essayer en ligne](https://loicgasser.github.io/tarot-marseille/)**

## ✨ Fonctionnalités

- **22 Arcanes Majeurs** du Bateleur au Monde, avec design CSS élégant (numéros romains, symboles, couleurs uniques)
- **Tirage 4 cartes** avec positions thématiques :
  1. 🔵 La Situation (Présent)
  2. 🔴 L'Obstacle ou l'Appui
  3. 🟣 Le Conseil ou l'Action
  4. 🟢 Le Résultat (Futur)
- **Cartes droites et renversées** (40% de chance d'être renversée, indication visuelle)
- **Interprétations détaillées** en français, adaptées à chaque position et orientation
- **Synthèse du tirage** prenant en compte les interactions entre les cartes
- **Animations** : mélange, retournement séquentiel, effets de lueur dorée
- **Responsive** : fonctionne sur mobile et desktop
- **100% client-side** : aucun backend, aucune API, aucune donnée collectée

## 🎨 Design

- Thème mystique sombre avec accents dorés
- Typographies Cinzel et Cormorant Garamond
- Fond étoilé animé
- Cartes avec bordures ornementées et effets de shimmer

## 🛠️ Stack technique

- **Zéro dépendance** : HTML + CSS + JavaScript vanilla
- Fichier unique `index.html` (~1000 lignes)
- Hébergé sur GitHub Pages
- Pas de build, pas de npm, pas de framework

## 📖 Symbolisme

Les interprétations suivent la tradition du Tarot de Marseille. Chaque carte possède :
- Une signification droite et une signification renversée
- Des interprétations contextuelles selon sa position dans le tirage
- Des symboles et couleurs fidèles à la tradition (couleurs chaudes pour les cartes positives, froides pour les cartes de défi)

## 🚀 Déploiement

Le site est déployé automatiquement via GitHub Pages depuis la branche `master`.

Pour une utilisation locale :
```bash
git clone https://github.com/loicgasser/tarot-marseille.git
cd tarot-marseille
open index.html  # ou tout serveur HTTP local
```
